# AI Appointment Booking System – Telegram Bot

## Overview
This project is a fully automated appointment booking system built
using n8n and Telegram. It handles user conversations, checks calendar
availability, creates confirmed bookings, logs data, and sends
confirmation emails automatically.

## Tools & Integrations
- n8n
- Telegram Bot API
- Google Calendar API
- Google Sheets API
- Email Automation
- REST APIs & Webhooks

## Workflow Summary
1. User interacts with the bot on Telegram
2. Bot collects required information step by step
3. Calendar availability is checked
4. Appointment is created after confirmation
5. Data is logged and emails are sent automatically

## Proof of Work
Real Telegram conversations and integration outputs are provided
in the screenshots folder. Personal data is anonymized.
